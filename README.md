## 소개
- CraftsMan은 전기 시퀀스 회로를 직접 구성하고 동작을 확인할 수 있는 교육용 시뮬레이션 프로그램입니다.
* **운영체제(OS):** Windows
* **권장 해상도:**
    * 1920 x 1080 (FHD)
    * 2560 x 1440 (QHD)
    * 3840 x 2160 (4K)

## 주요기능
- 승강기기능사 및 전기기능사 시뮬레이션
- 자동 배관 및 배선작업
- 부하가 병렬로 구성된 회로에서 동작
- 사용가능 입출력 기기: 푸쉬버튼스위치, 셀렉터스위치, 램프, 부저, 3상모터(Y-D 지원)
- 사용가능 동작검사기기: EOCR, 전자접촉기, 릴레이, 타이머, 플리커, 플로트레스
- 기타기기: 벨테스터기, 배선용차단기, 퓨즈홀더, 단자대, 배관 등
- 개발중인 기능(V0.5.0으로 배포예정)
  - 리미트스위치
  - 플로트레스 수통
  - 타이머 및 플리커 시간조절
- 수정중인 버그
  - 배관을 통과하는 전선 제거 후 실행취소하면 전선이 배관 밖으로 삐져나오는 현상
