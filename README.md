## CraftsMan(전기 시퀀스 시뮬레이터) 개요
- **프로그램 소개**
  - 전기 시퀀스 회로를 직접 구성하고 동작을 확인할 수 있는 교육용 시뮬레이션 프로그램입니다.
  - 실제 작업환경을 기반으로 승강기기능사 및 전기기능사 실기 연습이 가능합니다.

## 프로그램 정보
* **권장 운영체제(OS)**
    * Windows
* **권장 해상도**
    * 1920 x 1080 (FHD)

* **용량**

## 다운로드 및 실행방법
- 우측 **Releases** 탭에서 CraftsMan 최신 버전 압축파일(zip)을 다운로드하세요.
- 압축을 해제한 후 CraftsMan 파일을 실행하세요.
## 주요기능
- 승강기기능사 및 전기기능사 실기시험 시뮬레이션
- 자동 배관 및 배선작업
- 부하가 병렬로 구성된 회로에서 동작
- 사용가능 입출력 기기: 푸쉬버튼스위치, 셀렉터스위치, 램프, 부저, 3상모터(Y-D 지원)
- 사용가능 동작검사기기: EOCR, 전자접촉기, 릴레이, 타이머, 플리커, 플로트레스
- 기타기기: 벨테스터기, 배선용차단기, 퓨즈홀더, 단자대, 배관 등
- 개발중인 기능(V0.5.0으로 배포예정)
  - 리미트스위치
  - 플로트레스 수통
  - 타이머 및 플리커 시간조절
- 개선요청 사항
  - 사각박스 커버 앞면 뒷면 시각적 구분 필요
